Abstract

The Audiovisual Ontology is an OWL 2 QL ontology modelling the audiovisual domain (movies, TV series, shorts and related media), originally derived from the IMDb relational dataset and generalised so it can be reused beyond IMDb as a shared conceptualisation. It declares ten pairwise-disjoint domain classes, nine object properties and twenty-one data properties, including a reified Participation class that captures the ternary relationship between a Talent, a Title, and the Role/Category under which the talent participates. The ontology is intended as the schema layer of a knowledge graph: it is populated through an R2RML mapping from the IMDb database, validated against a SHACL shapes file, and exposed via a SPARQL endpoint. Its primary use cases are multi-criteria catalogue search and the analysis of historical and collaborative patterns in cinema (career timelines, recurring acting/directing pairs, genre evolution).
